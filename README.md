**Instructions**
In order to test our site, you will first need to download mySQL onto your computer. Then, you will need to copy all of the code in our NextHomeSchema.sql file and paste it into mySQL in order to load our database. Once that is done, you are ready to launch the website!

To launch the website, you can follow these simple steps: 
1. In your terminal, navigate to the directory where you have stored all of the project files. 
2. Once there, type in "nodemon app.js" in order to launch the code.
3. Now, you can go to your browser. Type in "localhost:3000/html/index.html"
4. You should now be on the homepage of our website!

Once in our website, it is best to create in account in order to use all of its functionalities.To do this, simply go to the nav bar at the top of the website and click the button to create an account. There, you can type in your name, email, and password. Now, you will be able to post reviews, and post listings. If you type in an email which is alright in user, the page will refresh.
